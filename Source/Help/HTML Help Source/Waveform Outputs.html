Waveform Outputs

This section contains all the available host to target DMA channels of the supported data type (SGL) in the FPGA bitfile.

What do you want to do?

Add a DMA Channel

Select the channel or channels you would like to add from the Available DMA list. Then select Add Selected.

Remove a DMA Channel

Select the channel or channels you would like to remove from the Available DMA list. Then select Remove Selected.
