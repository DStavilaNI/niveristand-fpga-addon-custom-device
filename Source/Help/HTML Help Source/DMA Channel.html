DMA Channel

This section defines the properties of the DMA channel. This information is used to parse the data out of the DMA channel and convert it to Veristand waveforms.

What do you want to do?

Define the DMA properties

To define the DMA properties, first enter in the Number of Channels being passed to the DMA channel. Then enter the period used to update the DMA channel on the FPGA. Next, set the Update Rate to define how often VeriStand services the DMA and updates the VeriStand waveform. Last, select Apply. As an example, if you have 8 analog input channels on the FPGA wired to a build array that is wired to a DMA channel in a for loop contained in a while loop that runs at 10 kHz, set 8 channels with a dt of 100 microseconds.

Output a Waveform

Most use cases involve acquiring waveforms. In the case of generating a waveform output with the FPGA add-on, a linkage must be created to a source waveform. To do this, go to the desired FPGA add-on output waveform channel and browse to the source waveform.
